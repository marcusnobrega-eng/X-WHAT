How X-WHAT Works

🔵

1. Draw on Canvas

Use the web-based Network Builder to place junctions, reservoirs, pipes, and loops. Set demands, elevations, diameters, and boundary conditions. Export a .xwhat.xml package.

📋

2. Run in Excel

Open X-WHAT_Model_V1.5.xlsm, import your package, and run the GRG Nonlinear Solver. Be sure to check the 'Roadmap' guide. Save the solved workbook.

📊

3. View Results

Upload the solved workbook to the Results Viewer. See pipe flows, node pressures, and a full cost breakdown (pipelines + pumping + tank + foundation) in an interactive dashboard.
